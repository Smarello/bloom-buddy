## User Story

**Epic:** EP-006 — Collezione Personale e Storico Analisi
**Priorità:** MEDIUM | **Story Points:** 2

**Story**
Come Giulia,
voglio vedere tutte le analisi passate di una pianta salvata,
così da tracciarne l'evoluzione dello stato di salute nel tempo.

**Criteri di Accettazione**

- [ ] La pagina di dettaglio pianta mostra tutte le analisi ordinate per data (dalla più recente)
- [ ] Ogni analisi mostra: data, foto thumbnail e indicatore di stato di salute (verde/giallo/rosso)
- [ ] L'utente può aprire il dettaglio completo di ogni analisi passata (diagnosi e consigli)

---

## Soluzione Tecnica

L'approccio consiste nel raggruppare le analisi esistenti per nome della pianta (`datiAnalisi.nomeComune`) e creare una pagina di dettaglio pianta con rotta `/collezione/[nomePianta]` che mostra lo storico cronologico. Non è necessario un nuovo modello Prisma: il raggruppamento viene derivato dal campo JSON `nomeComune` già presente in ogni `Analisi`.

- La pagina collezione (`/collezione`) viene modificata per raggruppare le schede per pianta, mostrando per ciascuna l'analisi più recente e un conteggio delle analisi totali
- Una nuova pagina `/collezione/[nomePianta]` mostra tutte le analisi di quella pianta in ordine cronologico discendente, con thumbnail, data e indicatore di salute
- Una nuova API route `/api/collezione/pianta/[nomePianta]` filtra le analisi dell'utente per `nomeComune` (case-insensitive) e le restituisce ordinate per data
- Il click su una singola analisi nello storico naviga alla pagina `/analysis?id={idAnalisi}` già esistente

## Strategia di Test

I test coprono il filtraggio corretto delle analisi per pianta, l'ordinamento cronologico e la protezione dell'accesso.

- Test API route `/api/collezione/pianta/[nomePianta]`: verifica filtro per nome pianta, ordinamento desc, autenticazione richiesta, risposta vuota per pianta inesistente
- Test rendering pagina storico: verifica visualizzazione di più analisi con stati diversi, ordinamento corretto, link al dettaglio analisi
- Test raggruppamento collezione: verifica che piante con lo stesso nome vengano raggruppate correttamente
